欢迎来到校园在线考试系统

教师使用帮助

  • 1.新增学生

  • 2.创建试卷

  • 3.查看学生成绩

  • 4.查看考试分析

  • 1. 您可以查看自己所带的班级和学生，可以根据课程名来查看选了该课程的学生，也可以根据专业/年级/班级来筛选学生
  • 2. 您可以查看自己所带的课程，并把课程添加到某个班级
  • 3. 您可以查看/添加/删除/修改课程的考试和试题，并把考试添加到某个班级，可以在已有考试中查看试卷
  • 4. 您可以查看学生成绩和试卷，还可以查看考试分析
  • 5. 您可以查看个人信息和修改密码

学生使用帮助

  • 1.查看待考试

  • 2.开始考试

  • 3.查看成绩

  • 4.查看个人信息

  • 1. 您可以查看自己的课程和授课教师
  • 2. 您可以查看自己的考试信息，查看已参加考试的成绩及试卷评判
  • 3. 您可以查看带参加的考试，并开始考试
  • 4. 您可以查看个人信息和修改密码
